# Ideia do Projeto – ColetaFácil

## Contexto
Muitas comunidades não têm acesso fácil a informações ou serviços para descarte correto de resíduos não comuns.

## Solução
Criar um aplicativo que conecte comunidades a empresas especializadas em coleta, reutilização ou descarte correto desses resíduos.

## Impacto esperado
- Redução de descarte irregular
- Incentivo à reciclagem
- Conscientização ambiental
